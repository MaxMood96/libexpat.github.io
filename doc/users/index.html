Be encouraged to join improving this website. Thank you!

NOTE: For bindings and 3rd-party wrappers, please check the dedicated page.

To be included in this list, the related piece of software needs to be Open Source (so that users can study its use of Expat) and notable, e.g. be packaged in multiple (unrelated) distributions of GNU/Linux.
